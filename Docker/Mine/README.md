# 🕯️ Minecraft Server (Vanilla)

## 🗺️ Estructura de Directorios

```text
.
├── mc-data/    # Aquí es donde se guardarán los datos del mundo
└── server-vanilla.yml
```

---

## 🚀 Despliegue

1. Crear el directorio `mc-data` en caso de que no exista.
2. Ejecutar el siguiente comando:

```bash
docker-compose -f server-vanilla.yml up -d
```
